                # 拓扑排序

                # 问题背景

                207. 课程表open in new window

                你这个学期必须选修 numCourses 门课程，记为 0 到 numCourses - 1 。
在选修某些课程之前需要一些先修课程。 先修课程按数组 prerequisites 给出，其中 prerequisites[i] = [ai, bi] ，表示如果要学习课程 ai 则 必须 先学习课程  bi 。

例如，先修课程对 [0, 1] 表示：想要学习课程 0 ，你需要先完成课程 1 。
请你判断是否可能完成所有课程的学习？如果可以，返回 true ；否则，返回 false 。

示例 1：
输入：numCourses = 2, prerequisites = [[1,0]]
输出：true
解释：总共有 2 门课程。学习课程 1 之前，你需要完成课程 0 。这是可能的。

提示：
1 <= numCourses <= 105
0 <= prerequisites.length <= 5000
prerequisites[i].length == 2
0 <= ai, bi < numCourses
prerequisites[i] 中的所有课程对 互不相同

                # 思路

                当完成一件事情需要先后顺序的时候，比如A之后才能做B，C之后才能做B.
                那么B的前面就有2个依赖 A 和 C，当A完成之后，B的前面只有C了；当C完成之后，B前面就没有了， 就可以完成B。
                这个就是拓扑排序的思路：使用数组记录每一个节点的前面有多少个依赖，如果没有依赖那可以直接处理， 当处理完一个节点之后，依赖这个节点的事情就可以减去一个依赖项，如果刚好减到没有依赖了，那么也可以处理了。 可以通过队列保存可以直接处理的点，当队列为空的时候，处理结束。

                # 代码实现

                //就这个题目来说，入参有`numCourses`课程总数，`prerequisites` 表示课程依赖
public boolean canFinish(int numCourses, int[][] prerequisites) {
    //拓扑排序
    //先要构造拓扑。然后找到入度为0的课
    int[] indrgree=new int[numCourses];
    //这个是题目需要，一般项目不需要，用于保存某一个科目的依赖科目，可能有很多，所以选择列表
    List<List<Integer>> edge=new ArrayList<>();
    for(int i=0;i<numCourses;i++){
        //初始化列表
        edge.add(new ArrayList<>());
    }

    for(int[] temp: prerequisites){
        //保存每个科目的依赖数目
        indrgree[temp[0]]++;
        edge.get(temp[1]).add(temp[0]);
    }

    Queue<Integer> queue=new ArrayDeque<>();
    for(int i=0;i<numCourses;i++){
        if(indrgree[i]==0){
            //保存没有依赖的科目
            queue.offer(i);
        }
    }
    int index=0;
    while(! queue.isEmpty()){
        index++;
        //取出没有依赖的科目
        int cur=queue.poll();
        //循环依赖这个科目的列表，当处理完cur,列表中的每一个科目的依赖数都减1
        for(int v: edge.get(cur)){
            indrgree[v]--;
            if(indrgree[v]==0){
                queue.offer(v);
            }
        }
    }
    return index==numCourses;

}
